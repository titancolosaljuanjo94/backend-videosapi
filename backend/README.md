# Backend UTP - version1.abril-26

Este proyecto es el backend para la aplicación UTP. Incluye los archivos principales `main.py`, `api.py` y un archivo de dependencias `requirements.txt`.

## Estructura del proyecto
- `main.py`: Archivo principal de arranque.
- `api.py`: Lógica de la API.
- `requirements.txt`: Dependencias del proyecto.
- `outputs/`: Carpeta para archivos de salida generados.

## Requisitos
- Python 3.8+
- Se recomienda usar un entorno virtual (`.venv`)

## Instalación
```bash
python -m venv .venv
source .venv/bin/activate  # En Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Ejecución
```bash
python main.py
```

## Notas
- Recuerda mantener actualizado el archivo `.gitignore` para evitar subir archivos innecesarios.
- Los archivos de salida se almacenan en la carpeta `outputs/`.
